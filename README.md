Project Overview
This project focuses on employee performance mapping for ScienceQtech using MySQL-based data analysis.The goal is to support the HR department in evaluating employee performance, identifying training needs, and improving promotion and compensation decisions through structured, database-driven insights.

Objective
To assist HR teams in:
Evaluating employee performance
Aligning salaries with organizational benchmarks
Estimating performance-based bonuses
Planning training and development initiatives using SQL-driven analysis

Tools & Technologies
MySQL
SQL Queries
Relational Data Modelling

Key Activities
Queried and analyzed employee salary data and job roles to ensure alignment with organizational benchmarks
Calculated performance-based bonuses using structured SQL queries
Estimated training cost projections based on employee performance levels
Generated structured insights to support promotion planning and employee development

Methodology
Designed relational tables for employee, role, salary, and performance data
Wrote optimized SQL queries for data extraction and analysis

Key Insights
Identified salary discrepancies across roles and performance bands
Highlighted high-performing employees eligible for bonuses and promotions
Provided data-backed recommendations for targeted training investments

Results
Improved HR decision-making through clear, database-driven performance mapping
Enabled transparent evaluation of employee performance and compensation
Supported structured promotion and workforce development planning

Applied conditional logic for bonus calculation and training cost estimation

Aggregated results to support HR decision-making
